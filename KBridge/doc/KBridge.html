KBridge: bridge to Cassiopee Solver

Version: 2.6 (03/2018)

Author: Onera

Preamble

KBridge is a bridge to Cassiopee Solver objects.

To use it with the Converter array interface, you must import the KBridge module:
import KBridge
Then, in the following, a is an array, and A a list of arrays.
To use it with the pyTree interface, you must import the module:
import KBridge.PyTree as KBridge
Then, in the following, a is a zone node and A is a list of zone nodes or a complete pyTree.
