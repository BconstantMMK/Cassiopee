Preamble

To use the Distributor module, you have to load it:
from elsA import *
import Distributor

Automatic load balance with Cassiopee Kernel

A simple tool is provided in *Cassiopée* to distribute automatically the meshes over the processors (without splitting). No node specification is required in the python file. You just have to add at the beginning of python file:
import Distributor
Distributor.register()

Important note: Every mesh must be submitted just after definition.
