Purpose: a set of python modules for pre- and post-processing of CFD computations [more].
 Authors: ONERA [more].

Reference documentation 4.0:
  • Converter: Data conversion module
    • Converter.Internal: pyTree basic management module
    • Converter.Filter: partial reading/writing of files
    • Converter.Mpi: parallel management of pyTrees
    • Converter.elsAProfile: adaptation for elsA solver
  • Geom: Geometry definition module
    • Geom.IBM: Specific geometry operations for IBM simulations
  • Transform: Generic mesh transformation module
  • Generator: Mesh generation module
  • Intersector: Mesh intersection module
  • Connector: Mesh connectivity module
    • Connector.IBM: Specific connectivity operations for IBM simulations
  • Initiator: Solution initialization module
  • Post: Solution post-processing and solution extraction
    • Post.ExtraVariables2: compute derived fields from primitive variables
    • Post.Rotor: Specific rotor and propeller post-processing
    • Post.IBM: Specific post-processing for IBM simulations
  • Dist2Walls: Wall distance computation
  • RigidMotion: Rigid motion management
  • Compressor: Compression module
  • Distributor2: Distribute blocks over processors
  • OCC: Interface to open-cascade for CAD
  • CPlot: Light plotter
    • CPlot.tkCassiopee: A Tk interface to module functions
    • CPlot.Decorator: Decorate CPlot images with matplotlib
